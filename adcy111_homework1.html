مراجعة أسئلة الشبكات (10 أسئلة)

اضغط على عنوان السؤال لعرض الخيارات والإجابة الصحيحة

السؤال 1: Hubs vs Switches
Which statement is true regarding Hubs and Switches?
  • Hubs are more efficient than switches for large networks
  • Hubs improve network security by isolating traffic
  • Hubs are largely obsolete and have been replaced by switches due to inefficiency and security issues
  • Hubs can only connect two devices at a time
الشرح: أجهزة الـ Hub ترسل البيانات لجميع الأجهزة (Broadcast) مما يسبب بطئاً ومشاكل أمنية، بينما الـ Switch ذكي ويرسل البيانات للجهاز المقصود فقط، لذا أصبح الـ Hub قديماً جداً.
السؤال 2: Star Topology
In a star topology, how are devices connected?
  • Every device connects directly to every other device
  • Devices connect to a central backbone cable only
  • All devices connect in a daisy chain without a central device
  • Each device has a dedicated connection to a central hub, switch, or router
الشرح: في تصميم النجمة (Star)، كل جهاز يتصل بكابل خاص ومستقل بنقطة مركزية واحدة (Switch أو Hub)، مما يسهل عزل الأعطال.
السؤال 3: Port Definition
Which statement best describes a 'port' in TCP/IP networking?
  • It determines the next hop in an IP routing path
  • It identifies a specific application or service on a device at the Transport layer
  • It translates domain names to IP addresses
  • It marks the physical location of a device on the network
الشرح: المنفذ (Port) هو رقم يُستخدم لتوجيه البيانات إلى التطبيق الصحيح داخل الجهاز (مثلاً: منفذ 80 للمتصفح، منفذ 25 للبريد) ويعمل في طبقة النقل.
السؤال 4: Full Mesh Topology
Which topology description matches a ‘full mesh’ network?
  • All nodes connect to a central hub only
  • Every node connects directly to every other node
  • Nodes are connected in a ring without a central hub
  • Some nodes connect to all others while some connect to a few
الشرح: في الشبكة الكاملة (Full Mesh)، يتصل كل جهاز بأسلاك مباشرة مع *جميع* الأجهزة الأخرى لضمان عدم انقطاع الخدمة (Fault Tolerance).
السؤال 5: Transport Layer
Which OSI layer is responsible for accurate delivery of data between two hosts, including reliability and flow control?
  • Data Link Layer
  • Application Layer
  • Physical Layer
  • Transport Layer
الشرح: طبقة النقل (الطبقة الرابعة) هي المسؤولة عن موثوقية البيانات (Reliability) والتحكم في التدفق (Flow Control) باستخدام بروتوكولات مثل TCP.
السؤال 6: IPv4 vs IPv6
What is the main difference between IPv4 and IPv6 addresses?
  • IPv4 is used only for LANs and IPv6 only for WANs
  • IPv4 addresses never change after assignment unlike IPv6 addresses
  • IPv4 is text-based and IPv6 is binary-based
  • IPv4 uses 32-bit addresses, while IPv6 uses 128-bit addresses
الشرح: الفرق الجوهري هو حجم العنوان؛ IPv4 محدود (32 بت) مما أدى لنفاد العناوين، بينما IPv6 ضخم جداً (128 بت) لخدمة عدد لا نهائي من الأجهزة.
السؤال 7: Switch Function
What is the primary purpose of a switch in a Local Area Network (LAN)?
  • To convert electrical signals to light signals for transmission
  • To manage internet routing between different networks
  • To forward frames to the correct destination port using MAC addresses
  • To replace the need for hubs entirely in all networks
الشرح: وظيفة المبدل (Switch) الأساسية هي قراءة عنوان الـ MAC وتوجيه إطار البيانات (Frame) للمنفذ المتصل بالجهاز المستهدف فقط.
السؤال 8: Data Communications
What does “data communications” refer to in networking terms?
  • The process of writing code for network devices
  • The physical installation of cables in a building
  • The exchange of data between two devices via a transmission medium
  • The act of converting data into electrical signals only
الشرح: اتصالات البيانات هي العملية الكاملة لتبادل المعلومات بين مرسل ومستقبل عبر وسيط ناقل (سلكي أو لاسلكي).
السؤال 9: MAC Addresses
Which statement describes MAC addresses?
  • They are IP addresses assigned by network administrators
  • They identify a NIC on the local network and are used at the Data Link layer
  • They change frequently during each hop in a WAN
  • They identify a device across multiple networks and are used at the Network layer
الشرح: عنوان الـ MAC هو البصمة الفيزيائية لبطاقة الشبكة (NIC)، ويستخدم للتعريف بالجهاز داخل حدود الشبكة المحلية (LAN) فقط.
السؤال 10: Logical Address (IP)
Which type of address is primarily used to identify a device's location across interconnected networks?
  • Domain name
  • Physical address (MAC)
  • IP address (Logical address)
  • Port address
الشرح: عنوان الـ IP (العنوان المنطقي) هو المسؤول عن توجيه البيانات عبر الشبكات المختلفة (Interconnected Networks) والإنترنت، بعكس الـ MAC المحلي.
